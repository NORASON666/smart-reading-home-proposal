# 智慧閱讀平臺 — 首頁版型提案（3 版本）

為 **2026-06-22 線上會議**準備，協助臺北市智慧閱讀平臺（凌網建置案）討論首頁版型方向。

> 麗湖國小老師反映「首頁的資訊都很重要」，Rebecca 也覺得不好取捨。
> 這份提案不是把所有區塊塞進一頁，而是用三種不同的**資訊架構策略**，
> 讓大家具體比較、當場做決定。

## 🔗 線上預覽

👉 **https://norason666.github.io/smart-reading-home-proposal/**

（從 `index.html` 進入，可切換三個版本）

## 三個版本

| 版本 | 策略 | 特色 |
|------|------|------|
| **A 每月主打** | 焦點領頭、其餘收斂 | 海洋主題大圖 Hero ＋兩大入口卡，最清爽 |
| **B 雜誌策展** | 館員精選當封面故事 | 襯線標題、留白多，**最精緻** |
| **C 雙欄分流** | 閱讀（藍）／活動（金）左右並置 | 分流但同框，首頁一眼看到兩邊 |

共同設計原則：

1. **首頁分流，不平鋪** — 電子書世界與活動世界各自有專區，首頁只當「入口大廳」＋本月焦點。
2. **精緻＝克制** — 大留白、克制陰影、拉大字級對比。
3. **讓你親手取捨** — 每版右下角 **⚙ 自由拼版面**，即時開關區塊、顯示剩幾塊，取捨一目了然。

## 開會這樣用

1. 三版輪流打開，請麗湖老師先憑直覺挑「順眼」的一版。
2. 點右下角 **⚙ 自由拼版面**，把覺得「首頁不必有」的區塊關掉——面板即時顯示剩幾個區塊。
3. 三版都用同一套**凌網 design-tokens**（HyVue CMS 1.1），結論可直接回饋給凌網當設計方向。
4. 每頁左下角 **A / B / C** 可快速切換對照。

## 關於圖片

- **YouTube 影音為真實內容**：「我是小主播」「小小說書人」皆連到[臺北市兒童深耕閱讀教育網官方頻道](https://www.youtube.com/channel/UCIxfcH9pe9JJi1PtRJG9NZw)的實際影片，點下去就能看。
- 書封、情境照為**擬真示意圖**（主題授權圖庫），書名、活動、公告皆為示意假資料。
- 站名「臺北市智慧閱讀網」為暫定 placeholder。
- 真正的拖拉式版面管理屬凌網後台交付範圍；本提案以「比較具體版本 ＋ 開關區塊」協助決策。
