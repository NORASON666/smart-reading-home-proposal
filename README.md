# 智慧閱讀平臺 — 首頁版型提案

為 **2026-06-22 線上會議**準備，協助臺北市智慧閱讀平臺（凌網建置案）討論首頁版型方向，
提供 **資訊教育科、國教科** 老師參考選擇。

> 麗湖國小老師反映「首頁的資訊都很重要」，Rebecca 也覺得不好取捨。
> 這份提案不是把所有區塊塞進一頁，而是用兩種不同的**資訊架構方向**，
> 讓大家具體比較、當場做決定。

## 🔗 線上預覽

👉 **https://norason666.github.io/smart-reading-home-proposal/**

## 兩個版本

| 版本 | 方向 | 重點 |
|------|------|------|
| **A 內容優先**（建議給老師參考） | 書單／學生創作／AI 推薦為主，活動／成果為輔 | 首頁聚焦「閱讀內容」，沒有大型入口卡，Banner 不放統計數字 |
| **B 雜誌策展** | 館員精選封面故事 ＋ 兩大入口 | 編輯式雜誌版面、最精緻；保留「📚 電子書館／🎯 活動專區」兩大入口 |

## 收錄的內容區塊

- **書單**：本週選書、主題書展、AI 為你推薦
- **學生創作**：學生自製電子書（標「創」併入館藏）
- **以書映光（國中深耕閱讀）**：國中「以書映光」閱讀挑戰任務投稿（本期讀本《動物眼中的世界》）＋ **本期優良作品**展示（學生姓名／學校／讀本）
- **推廣活動**：每月專題任務、我是小主播、小小說書人、線上讀書會、自編故事劇本、閱冠磐石獎、捕捉閱讀紀錄片、多文本讀報
- **閱讀專欄（閱讀私房菜）**、**閱讀成果展示**、**公告**、**好站連結**

## ⚙ 自由拼版面

每版右下角的 **⚙ 自由拼版面** 面板可以：

- **勾選／取消** → 即時顯示或隱藏區塊（面板顯示目前剩幾個區塊）
- **拖曳 ⠿** → 用滑鼠調整區塊的上下順序
- **全部還原** → 回到原始版面

讓老師當場「拼」出想要的首頁，直接看出取捨。

## 開會這樣用

1. 兩版輪流打開，請老師先憑直覺挑「順眼」的方向。
2. 用 ⚙ 面板開關 / 拖曳排序，當場拼出理想首頁。
3. 兩版都用同一套**凌網 design-tokens**（HyVue CMS 1.1），結論可直接回饋給凌網當設計方向。
4. 每頁左下角 **A / B** 可快速切換對照。

## 關於圖片

- **YouTube 影音為真實內容**：「我是小主播」「小小說書人」皆連到[臺北市兒童深耕閱讀教育網官方頻道](https://www.youtube.com/channel/UCIxfcH9pe9JJi1PtRJG9NZw)的實際影片。
- 書封、情境照為**擬真示意圖**；書名、活動、公告、投稿任務皆為示意假資料。
- **Banner 不放尚難取得的統計數字**（如參與人數）。
- 站名「臺北市智慧閱讀網」為暫定 placeholder。真正的拖拉式版面管理屬凌網後台交付範圍。
